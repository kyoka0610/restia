# 仲良し子 (Nakayoshiko)

このプログラムは、カップルや親友同士で記念日やプロフィールを共有するためのダッシュボードアプリです。

## 🛠️ 技術スタック・使用言語
- **プログラミング言語**: Python 3.x
- **Webフレームワーク**: Streamlit
- **データ保存**: JSON形式のローカルファイル (`data.json`)

## 📦 必要なモジュール・ライブラリ
このアプリを動かすには、以下の外部ライブラリが必要です。

- `streamlit`: WebUIを構築するためのフレームワーク
- `Pillow`: 画像処理（アルバム機能での画像リサイズなど）を行うライブラリ

標準ライブラリ（インストール不要）:
- `json`: データの保存・読み込み
- `os`, `base64`, `calendar`, `datetime`, `io`

## 🚀 プログラムの動かし方・確認手順

### 1. 必要なライブラリのインストール
まず、ターミナル（コマンドプロンプトやPowerShell）を開き、アプリのディレクトリに移動します。
その後、以下のコマンドを実行して必要なライブラリをインストールしてください。

```bash
pip install streamlit pillow
```

### 2. アプリの起動
同じディレクトリ内で、以下のコマンドを実行してStreamlitアプリを起動します。

```bash
streamlit run restia.app.py
```

### 3. ブラウザでの確認
コマンドを実行すると、自動的にブラウザが立ち上がり、アプリの画面が表示されます。
もし自動で開かない場合は、ターミナルに表示されるURL（通常は `http://localhost:8501` ）をブラウザのアドレスバーにコピー＆ペーストしてアクセスしてください。

### 4. アプリの終了
アプリを終了してサーバーを停止するには、アプリを実行しているターミナル上で `Ctrl + C` を押してください。
